This project is made from `html, css and javascript`

## How to set up

Install `Visual Studio Code`

Install `Extension Live Server`

### First

Clone the project 
`git clone https://github.com/quyentruong2411/ktpmud_nhom12.git`

### Start the project

Navigate `index.html`
Right click, `Open with live server`
